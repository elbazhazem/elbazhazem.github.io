Cloud computing is a recent evolving technology that every organization nowadays wishes to apply to its company for greater profitability and scalability. The aim of this study is to identify critical issues and challenges in cloud computing for banking organizations. Indeed, we identified twenty critical cloud issues for banking organizations based secondary data such as Protection of Cloud Data and Privacy, authorization, and authentication for a cloud system, Safeguard the Confidentiality of Cloud Information, accessibility effectiveness or availability of cloud data, etc. Furthermore, we derived the critical issues from cloud computing models: Mobility and banking application, cloud services, cloud deployment, cloud risk management, and cloud security model. As future work, we intend to mitigate the critical issues depend on data qualitative and quantitative by using intelligence techniques. Successful essential identification of cloud security issues will significantly improve the probability of cloud banking success.
